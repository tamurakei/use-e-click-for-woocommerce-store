== Description ==

このプラグインは、Woocommerce を必要とします。
Woocommerce の注文完了画面(Thank you page)に、イークリック(https://e-click.jp/)のトラッキングコードの表示を追加します。

利用の前には、管理画面にてイークリックより発行されるID の登録が必要です。

== Installation ==

1. Upload the entire `` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Set Advitizer ID issued by e-click.jp at `/wp-admin/admin.php?page=use-affiliate-sevice-for-woocommerce-store.php`
